# dotfiles
my personal dotfiles

## Clone

```bash
git clone git@github.com:catnstein/dotfiles.git ~/.dotfiles
```

## Symlinks

- zsh
```bash
ln -s ~/.dotfiles/zshrc ~/.zshrc
```

### .config

```bash
mkdir ~/.config
```

- alacritty
```bash
ln -s ~/.dotfiles/config/alacritty ~/.config
```

- nvim
```bash
ln -s ~/.dotfiles/config/nvim ~/.config
```
